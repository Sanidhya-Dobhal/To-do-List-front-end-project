To-do List

About

Never forget a thing! This is an interactive To-do list/Task Manager which lets you add tasks to be done to the list through the task window. The tasks are arranged chronologically in the order you add them.
You have to add the task name, accompanied by an optional task description. You can also categorise the task in one of the 3 provided categories.
Once the task has been completed simply mark it as "done" and watch it vanish from your list.
Stay Organised 👍 !!
